# Orchestrix MVP Plan

## Completed Phases

### Phase 1: Project Scaffolding ✅
- electron-vite + Vue 3 + TypeScript

### Phase 2: Terminal Core ✅
- node-pty + xterm.js

### Phase 3: Multi-Pane Terminals ✅
- Up to 3 terminal panes
- Drag reorder
- Project binding

### Phase 4: Project Management + File Explorer ✅
- File tree with right-click context menu
  - New file
  - New folder
  - Rename
  - Delete
  - Move

### Phase 5: Split Code Editor ✅
- VS Code style drag-split
- Per-project layout persistence
- Cursor restore

### Phase 5b: Git Integration ✅
- File status badges (M/A/D/U) in file tree
- Blue hatched gutter marks for modified lines
- Inline blame on cursor line
- Click gutter for inline diff with revert
- Full diff view with draggable splitter

### Phase 5c: Media Preview + Hex Editor ✅
- Image/video/audio preview via base64 data URL
- Hex Editor for binary files with address/hex/ascii columns and byte inspector

### Phase 5d: Multi-Language Syntax Highlighting ✅
- C, C++, Java, Kotlin, C#, Objective-C, Shell, PowerShell, Go, Rust, Ruby, Swift, Lua, Perl, YAML, TOML, SQL, Dockerfile, CMake
- Plus existing: JS/TS/JSON/HTML/CSS/Vue/Python/Markdown
